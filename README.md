# 点歌盲盒 · 深夜雨窗

一场隔着起雾玻璃的点歌仪式。控制台负责导入歌单与设置，雨窗舞台页给观众看——抽歌时暖光擦开雾气，歌名从模糊里慢慢凝实。

**在线预览（GitHub Pages 开启后）：**  
https://zlxjkason720821.github.io/song-roulette/stage_display.html

---

## 文件说明

| 文件 | 用途 |
|------|------|
| `control_panel.html` | 控制台：导入歌单、选音色、管理待抽/已唱 |
| `stage_display.html` | 舞台页：抽取展示，给观众看 |
| `shared.css` | 共用样式（雨窗配色、面板、页脚） |
| `shared.js` | 共用逻辑（本地存储、音色引擎、CSV 解析） |

无需安装依赖，用浏览器直接打开 HTML 即可。

## 快速开始

### 方式一：本地使用

1. 下载或克隆本仓库
2. 双击打开 `control_panel.html`
3. 导入歌单后，点击「打开雨窗舞台」（或单独打开 `stage_display.html`）
4. 舞台页点「抽取」，或按 **空格键** 抽歌

### 方式二：双屏 / 双窗口（推荐）

```
┌─────────────────────┐     ┌─────────────────────┐
│   control_panel     │     │   stage_display     │
│   （操作员自己看）    │     │   （观众 / 投屏看）   │
└─────────────────────┘     └─────────────────────┘
          │                            │
          └──────── 浏览器本地存储同步 ──┘
```

- **控制台**放在自己屏幕上操作
- **舞台页**全屏投屏或第二块显示器
- 两边数据实时同步，无需刷新

## 控制台操作手册

### 1. 导入歌单

支持两种格式，切换顶部「纯文本 / CSV」标签即可。

**纯文本** — 一行一首歌名：

```
虫儿飞
晴天
后来
天空之城
```

**CSV** — 第一列为歌名，后两列可选（歌手、备注）：

```
歌名,歌手,备注
虫儿飞,郑伊健,降调
晴天,周杰伦,
后来,刘若英,粉丝点过
```

- 点击 **导入到池子** 写入待抽列表
- 也可点 **读取 .txt / .csv** 从文件导入
- 重复歌名会自动跳过，不会重复加入

### 2. 抽中音色

| 操作 | 说明 |
|------|------|
| 开关 | 点击铃铛图标，控制抽中时是否播放音效 |
| 音色 | 五种可选：空灵·星音、水晶、风铃、颂钵、清脆铃 |
| 试听 | 在控制台本地预览，方便挑选 |

音色设置会同步到舞台页，**实际抽中时的声音由舞台页播放**。

### 3. 歌单管理

| 按钮 | 作用 |
|------|------|
| 待抽列表 `✕` | 从池子中删除该首歌 |
| 已唱列表 `↺` | 把已唱的歌放回待抽 |
| 把已唱全部放回 | 已唱歌曲全部回到待抽池 |
| 清空全部 | 删除所有歌单（不可撤销） |

顶部数字会同步显示当前 **待抽** 与 **已唱** 数量。

## 舞台页操作手册

### 界面状态

| 状态 | 画面文案 | 说明 |
|------|----------|------|
| 待机（有歌） | 窗外有雨 | 等待抽取 |
| 待机（无歌） | 窗外有雨 | 需先在控制台导入歌单 |
| 抽取中 | 拭去窗上的雾 | 雾气变薄，暖光擦窗 |
| 抽中 | 为你唱这一首 | 歌名从模糊凝实，播放音色 |
| 唱完 | 雨停了 | 歌曲移入已唱列表 |

### 抽取后

抽中一首歌后，会出现两个选项：

- **这首唱了** — 移入已唱，继续抽下一首
- **放回池子** — 不算已唱，可重新抽

### 快捷键

| 按键 | 功能 |
|------|------|
| `空格` | 抽取（焦点不在输入框时） |

## 数据存储说明

- 歌单与音色存在浏览器 **localStorage**，键名 `songRoulette_v1` / `songRoulette_sound`
- 控制台与舞台页通过 `storage` 事件跨窗口同步
- 数据仅在本机浏览器中，清除站点数据会丢失歌单
- 换电脑或换浏览器需重新导入

## 常见问题

**Q：舞台页显示「在控制台填入歌单」？**  
A：先在 `control_panel.html` 导入歌曲，再回舞台页抽取。

**Q：两边数字不一致？**  
A：正常情况下约 1.5 秒内自动同步。若仍异常，刷新两个页面。

**Q：没有声音？**  
A：检查控制台音色开关是否打开；浏览器可能拦截自动播放，先在舞台页点一次「抽取」以激活音频。

**Q：投屏时想隐藏控制台？**  
A：只投屏 `stage_display.html` 即可，操作留在自己的屏幕上。

## 设计说明

视觉主题 **深夜雨窗**：深蓝灰夜色打底，昏黄灯光为唯一暖色。七层雨窗（雨云 → 雨丝 → 磨砂玻璃 → 雾气 → 水痕 → 灯光呼吸 → 内容），抽歌时暖光横向擦开雾气，歌名从模糊中凝实。
